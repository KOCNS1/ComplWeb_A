Exercice 05 - Events en situation plus réelle

Afin de consolider des évènements dans un cas d'usage plus "réel", vous aurez à charge ici de créer une gallerie d'images complètement interactif à l'aide de javascript.

> Ci-dessous une base de gallerie d'images en HTML + CSS complètement statique. Votre but est de rajouter les évènements suivants:
Déroulez pour voir les détails
  • Un click sur les miniatures change l'image principale
  • En dessous de l'image principale, il y a un petit bloc dédié à afficher une description de l'image. Cette description est présente dans l'attribut alt de l'image miniature. Vous devrez extraire ce texte afin de l'afficher à côté de l'image principale sélectionnée
  • Un click sur les flèches gauche et droite cyclent à travers les images
  • Il doit être possible de de cycler à travers les images en appuyant sur les flèches droite et gauche du clavier
  • La page devra cycler à travers les images automatiquement toutes les 5 secondes
  • Empêchez les images de cycler automatiquement lorsque l'utilisateur positionne sa souris sur l'image principale ou les miniatures.
  • Un clic sur l'image principale permet de zoomer l'image pour la voir couvrir toute la page et l'afficher "en plein écran"
