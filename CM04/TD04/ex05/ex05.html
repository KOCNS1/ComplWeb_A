Exercice 05 - Formulaire avec requêtes dynamiques

Utilisons l'API Fetch dans un formulaire pour faciliter la saisie d'une adresse

> Il existe des APIs gouvernementales officielles pour récupérer des informations géographiques sur le territoire français.
Vous devrez faire usage de ces webservices pour permettre une saisie dynamique du formulaire ci-dessous. L'utilisateur sélectionne dans un premier temps sa région. Suite à cette sélection, chargez la liste des départements. Faites de même pour les communes.
Utilisez les webservices suivants:
  • https://geo.api.gouv.fr/regions pour la liste des régions
  • https://geo.api.gouv.fr/regions/${codeRegion}/departements pour la liste des départements basé sur le code région fourni dans l'URL
  • https://geo.api.gouv.fr/departements/${codeDepartement}/communes pour la liste des communes basé sur le code département fourni dans l'URL
