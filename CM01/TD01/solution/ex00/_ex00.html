Exercice 00 - Hello world!

Dans le cadre de cet exercice 0, le but sera d'inclure nos premiers scripts javascript dans notre page et de se familiariser avec le fonctionnement de base

> Ajoutez votre premier fichier de script dans cette page html. Celle-ci devra afficher le texte "Hello world!" avec la fonction alert() en javascript
> Incluez le script que vous venez de créer...
  • Dans la balise HEAD de ce fichier HTML
  • En haut de la balise BODY
  • Au milieu de la balise BODY
  • A la fin de la balise BODY
Observez le résultat et formulez une hypothèse sur l'exécution de vos scripts et écrivez la réponse dans le code HTML de cette page ci-dessous
Réponse
Les scripts sont exécutés dans l'ordre qu'ils apparaissent dans le fichier HTML. Les scripts sont parsés séquentiellement dans la page HTML. Les scripts déclarés avant le contenu HTML de la page seront exécutés avant que le contenu HTML ne soit chargé
