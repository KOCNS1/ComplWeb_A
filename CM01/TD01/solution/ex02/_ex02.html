Exercice 02 - Tableaux et boucles

Dans cet exercice, nous allons réaliser notre première boucle en implémentant l'algo simple du "FizzBuzz".

> Le "FizzBuzz" est un algorithme simple dont le but est d'imprimer les chiffres allant de 1 à n. Si le chiffre est divisible par 3, on imprime "Fizz" ; si'l est divisible par 5, on imprime "Buzz" ; si'l est divisible par les deux, on imprime "FizzBuzz".
En pratique, l'algorithme devra imprimer cette séquence : 1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz, 16, 17, Fizz, 19, Buzz, Fizz, 22, 23, Fizz, Buzz, 26, Fizz, 28, 29, Fizz Buzz, 31, 32, Fizz, 34, Buzz, Fizz, ... Ecrivez une première version de cet algo en imprimant séquentiellement dans la console le résultat de ces chiffres allant de 1 à 100.
> Adaptez votre script pour demander à l'utilisateur jusqu'où il souhaite afficher les résultats. Prenez encore le soin de vérifier la saisie utilisateur.
> Adaptez encore votre script pour stocker le résultat de cet algorithme dans un tableau dans une seule variable. Loggez ce résultat dans la console.
