Exercice 05 (avancé) - Jeu du morpion

Cette fois, nous allons réaliser un jeu de morpion. Typiquement, le jeu du morpion se fait sur une grille de 3x3, mais dans cet exercice, nous allons laisser le choix de la taille de la grille à l'utilisateur, donc nous ne pouvons pas partir d'une base en HTML. Tout sera créé dynamiquement!

> Réalisez un jeu de morpion fonctionnel sur cette page sur une grille de 3x3 (règles du jeu ici au cas où : https://www.regles-de-jeux.com/regle-du-morpion/)
> Au lancement du jeu, demandez à l'utilisateur quelle taille de grille il veut. Ainsi, il pourra créer une grille de 3x3, 5x5, voir même 30x30 s'il est maso. Adaptez votre code pour répondre à ce besoin. Si vous avez conçu votre code initial dans la première partie de cet exercice, cette partie ne devrait pas être trop compliquée :)
